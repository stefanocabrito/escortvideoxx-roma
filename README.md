# Escort Roma - escortvideoxx.com

Sito web per escort a Roma, integrato con escortvideoxx.com.

## Caratteristiche

- ✅ Disclaimer 18+ con localStorage
- ✅ 7 profili escort con foto
- ✅ 4 categorie (Classiche, MILF, Mistress, Dominatrice)
- ✅ Design responsive (mobile + desktop)
- ✅ SEO optimizzato
- ✅ Colori brand (Magenta, Rosso, Grigio)

## Stack

- React 19
- Tailwind CSS 4
- TypeScript
- Vite

## Installazione

```bash
pnpm install
```

## Sviluppo

```bash
pnpm run dev
```

## Build

```bash
pnpm run build
```

## Struttura

```
client/
  public/images/     ← Foto delle modelle
  src/
    components/      ← Componenti React
    App.tsx          ← App principale
    main.tsx         ← Entry point
    index.css        ← Tailwind + stili globali
```
